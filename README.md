# UnityBot
Discord bot allowing verification of new users, role assignment and administration commands

## Features
- Add roles with buttons
- Presentation system

## Config
> .env file
```
CLIENT_ID = ""
GUILD_ID = ""
MEMBER_ROLE_ID = ""
OWNER_ID = ""
PRESENTATION_CHANNEL_ID = ""
TOKEN = "BOT TOKEN"
```
